== Description ==

[WP-PostViews](https://wordpress.org/plugins/wp-postviews/) enables you to display how many times a post/page had been viewed. This plugin allows you to reward post authors for each view they receive with myCred points.

= myCred for WP-PostViews Features =
Here are the most notable features of myCred for WP-PostViews.

* Award points for post views.
* Badge support.
* Translation ready.

= Plugin Requirements =

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)
* [WP-PostViews](https://wordpress.org/plugins/wp-postviews/)
* WordPress 5.0+
* PHP 5.3+

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for WP-PostViews
3. Find myCred for WP-PostViews and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
